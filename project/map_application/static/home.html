見車率

根據YouBike即時資訊回傳資料計算，見車率越高代表某站越少無車可借的情況。具體來說，見車率定義為某站當天若有正常營運，在06:00~23:59間站上至少有1台車的時間比例。90%以上為高見車率；60%~90%為中見車率；60%以下為低見車率
